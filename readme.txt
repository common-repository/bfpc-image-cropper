== Description ==
 
BFPC Image Cropper is good image cropping plugin that can give options for site visitor can edit image without login and without uploading images into you WordPress media library.

Usage as shortcode:
– Use Shortcode [bfpc_image_cropper] to set image cropper on your WP page or post.

FEATURES
* Allow to Rotate Image, Horizontal-flip and Vertical-flip image.
* Allow to Cropping specific area from original image.
* Save the Cropped image into png format.
* Upload new image and crop without page reload.
 
== Installation ==
 
Follow this following steps to start cropping: 
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add WP page or post with Shortcode [bfpc_image_cropper] and visit it.
